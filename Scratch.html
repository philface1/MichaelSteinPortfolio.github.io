Scratch Content

Name Assignment

Description

The Scratch Name Project was an assigment which introduced students to the fundamentals of sprites and the various tasks which could be assigned to them. In order to begin their actions, all that you need to do is click on the green flag.

Concepts Learned:

  • Using event blocks
  • Conditional statements
  • Using loops - forever blocks, repeat blocks
  • Sprite
  • Movement

Computer Buying Project

Description

The Scratch Computer Buying Project was an assignment which required students to obtain information regarding a specific computer they would recommend to someone looking to purchase a computer. Taking the customers preferences into thought, the group of students were tasked with finding a computer they found to be suitable. The project was then required to be presented before the class with a clear description of each device, and the overall choice of the best computer.

Concepts Learned:

  • Sprites interacting with one another

Final Project

Description

The Scratch Final Project was the final project completed by our class using the Scratch program. Given the task of creating either a game or a story, we were set to work with a little over a week to complete it. I chose to write a story, implementing blocks and messages to allow my sprites to interact with one another in order to produce a coherent and fluent production.

Concepts Learned:

  • Broadcasting messages
  • Loops to a greater extent
  • Incorporating a greater number of backgrounds
  • Establishing all units in their correct order
